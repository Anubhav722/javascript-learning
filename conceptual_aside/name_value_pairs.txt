Name value pair
A name which maps to a unique value

The nanme may be defined more than once, but only can have one value in any given context.
That value may be more name/value pairs.

OBJECT:
A collection of name value pairs.
The simplest defintion when talking about Javascript.

For ex.

address: {
		"street": "main",
		"number": 100,
		"apartment": {
			"floor": 3,
			"number": 301
			}
	}
